The CyberDrain Improved Partner Portal

CIPP helps Microsoft Partners manage multi-tenant administration — user management, standards deployment, and day-to-day tenant operations — from a single open-source portal. This is the project landing page.

Get started → View source
🔒

This page is informational only. The CIPP portal is self-hosted and sits behind Microsoft Entra ID (Azure AD) authentication with its own backend API. No tenant data, credentials, or the authenticated application are served from this static site.

Security

How this page is hardened

  • ✓ Strict Content-Security-Policy — script-src 'none': no JavaScript executes on this page, and all resources are locked to same-origin.
  • ✓ Fully self-contained — no external scripts, fonts, trackers, or network calls, so there is no third-party attack surface.
  • ✓ No referrer leakage — referrer: no-referrer and rel="noopener" on outbound links.
  • ✓ Hardened deploy pipeline — least-privilege GitHub Actions permissions, egress auditing, and pinned dependencies via Dependabot.
  • ✓ Machine-readable security contact — see /.well-known/security.txt (RFC 9116).
